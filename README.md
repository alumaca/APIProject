# APIProject
Sviluppo di un'API che traduce i comandi inviati dalla Terra in istruzioni comprensibili per un rover

## Configuration
Inastall Ruby on Rails : https://guides.rubyonrails.org/v5.0/getting_started.html

## Applicazioni
Ci sono due app: AppGenerateRoverDatas per inserire i dati e AppCalculateDatas che elabora i dati

##Fasi
1. Insttallare ruby
2. Installare Ruby on Rails
3. Scaricare il progetto APIPoject e scompattarllo
4. Da terminare collocarsi nella cartella /APIProject
5. Anare in /APIProject/AppGenerateRoverDatas e creare il Database con il comando
> rails db:crearte
6. Migrasre le tabelle  con il comando
> rails db:migrate
